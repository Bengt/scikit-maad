maad.sound.sound_func.load¶

maad.sound.sound_func.load(filename, channel='left', detrend=True, verbose=False, display=False, savefig=None, \*\*kwargs)[source]¶

Load a wav file (stereo or mono)

Parameters
filenamestring

The name or path of the .wav file to load if you want to extract the date of creation of the file, the filename must have this postfix : XXXX_yyyymmdd_hhmmss.wav with yyyy : year / mm : month / dd: day / hh : hour (24hours) / mm : minutes / ss : seconds

channel{‘left’, right’}, optional, default: left

In case of stereo sound select the channel that is kept

detrendboolean, optional, default is True

Subtract the DC value.

verboseboolean, optional, default is False

print messages into the consol or terminal if verbose is True

displayboolean, optional, default is False

Display the signal if True

savefigstring, optional, default is None

Root filename (with full path) is required to save the figures. Postfix is added to the root filename.

**kwargs, optional. This parameter is used by plt.plot and savefig functions
