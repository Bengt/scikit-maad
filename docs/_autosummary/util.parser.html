util.parser¶

Utilitary functions to parse or read files

Functions

date_from_filename

Extract date and time from the filename.

date_parser

Parse all filenames contained in a directory and its subdirectories Keep only filenames corresponding to extension Filenames must follow : - SM4 format (XXXX_yyyymmdd_hhmmss.wav) - or POSIX format (for audiomoth) The result is a panda dataframe with ‘Date’ as index and ‘File’ (with full path as column)

read_audacity_annot

Read audacity annotations file (or labeling file) and return a Pandas Dataframe with the bounding box and the label of each region of interest (ROI)

write_audacity_annot

Write audio segmentation to file (Audacity format)
