util.miscellaneous.nearest_idx¶

util.miscellaneous.nearest_idx(array, value)[source]¶

Find nearest value on array and return index

Parameters
array: ndarray

array of values to search for nearest values

value: float

value to be searched in array

Returns
idx: int

index of nearest value on array

Examples

>>> x = np.array([1,2,3])
>>> nearest_idx(x, 1.3)
    0
>>> nearest_idx(x, 1.6)
    1
