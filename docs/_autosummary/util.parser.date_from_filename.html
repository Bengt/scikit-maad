util.parser.date_from_filename¶

util.parser.date_from_filename(filename)[source]¶

Extract date and time from the filename. Return a datetime object

Parameters
filenamestring

The filename must follow this format : XXXX_yyyymmdd_hhmmss.wav with yyyy : year / mm : month / dd: day / hh : hour (24hours) / mm : minutes / ss : seconds

Returns
dateobject datetime

This object contains the date of creation of the file extracted from the filename postfix. The filename must follow this format : XXXX_yyyymmdd_hhmmss.wav with yyyy : year / mm : month / dd: day / hh : hour (24hours) / mm : minutes / ss : seconds
