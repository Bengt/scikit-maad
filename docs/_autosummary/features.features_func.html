High level functions for multiresolution analysis of spectrograms Code licensed under both GPL and BSD licenses Authors: Juan Sebastian ULLOA <jseb.ulloa@gmail.com> Sylvain HAUPERT <sylvain.haupert@mnhn.fr>

Functions

centroid

Computes intensity centroid of the 2D signal (usually time-frequency representation) along a margin, frequency (0) or time (1).

compute_rois_features

Computes shape and central frequency features from signal at specified time-frequency limits defined by regions of interest (ROIs)

create_csv

Create a .csv file containing all the features (shapes, centroids and labels)

filter_bank_2d_nodc

Build a Gabor filter bank with no offset component

filter_mag

Normalizes the image and computes im and real part of filter response using the complex kernel and the modulus operation

filter_multires

Computes 2D wavelet coefficients at multiple octaves/pyramids

gabor_kernel_nodc

Return complex 2D Gabor filter kernel with no DC offset.

get_features_wrapper

Computes shape of 2D signal (image or spectrogram) at multiple resolutions using 2D Gabor filters

opt_shape_presets

Set values for multiresolution analysis using presets or custom parameters

plot_shape

Plot shape features in 2D representation

save_csv

Create and save a .csv file containing all the features (shapes, centroids and labels)

save_figlist

Save a list of figures to file.

shape_features

Computes shape of 2D signal (image or spectrogram) at multiple resolutions using 2D Gabor filters

shape_features_raw

Computes raw shape of 2D signal (image or spectrogram) at multiple resolutions using 2D Gabor filters.
